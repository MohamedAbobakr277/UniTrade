## 2024-05-24 - Tooltips and ARIA Labels for Icon-Only Buttons\n**Learning:** Icon-only buttons without visual tooltips or ARIA labels hinder accessibility and user experience. Users, especially those relying on screen readers, need explicit context for actions. Wrapping `IconButton`s with Material UI `Tooltip` and adding `aria-label` provides this without altering existing styles.\n**Action:** Always wrap `IconButton` with `Tooltip` and ensure `aria-label` is set for all standalone icon buttons across the application.
